    • Estratégia de dados
    Todos os dados utilizados serão provenientes de instituições públicas e terão suas licensas respeitadas e todos os dados produzidos, já com um maior grau de riqueza, serão disponibilizados abertamente, respeitando as licensas dos dados que lhes deram origem.
    Governança de dados
    Os dados brutos que entrarão no sistema serão processados e os novos dados produzidos serão disponibilizados abertamente de forma estática na web, enquanto não existirem políticas para atualização. Fora a reestruturação que será promovida para a análise proposta, não há planos para cuidados adicionais com a base.
    Analítica de dados
    Toda a inteligência gerada será proveniente de indicadores extraídos dos dados reestruturados e processados, a fim de evidenciar relações emergentes do cruzamento das distintas bases que compõe os insumos do projeto. Não existem limitações técnicas ou analíticas aparentes que impessam ou inviabilizem o desenvolvimento do projeto.
    Visualização de dados
    Espera-se que o resultado indique, de forma objetiva, impacial e generalista, se partidos e/ou políticos promovem ou não o crescimento de indicadores socioeconômicos das regiões que os elegeram, servindo como um indicador tanto individualizado para os políticos eleitos, quanto comparativo para as diversas zonas eleitorais.
    Execução da decisão
    Na escasez de fontes imparciais e orientadas a dados no que diz respeito à esfera política, o projeto busca empoderar o Eleitor, iluminando seu caminho rumo a decisões mais inteligentes.
